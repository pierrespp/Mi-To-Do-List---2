# Mi To Do List — Rebuild Prompt

Crie um aplicativo web chamado “Mi To Do List”.

O app deve ser uma To-Do List moderna inspirada no Microsoft To Do, porém com estética extremamente Kawaii Pride / Queer Aesthetic premium.

O sistema é pessoal (single-user), simples, intuitivo e extremamente bonito visualmente. NÃO transformar em sistema corporativo complexo.

---

# Stack Obrigatória

* React + Vite
* TypeScript
* TailwindCSS
* Zustand
* Supabase

---

Quero POUCOS arquivos.

* excesso de componentes

* overengineering

Objetivo:

* código simples

# Funcionalidades Principais

## 1. Workspaces

Usar rotas dinâmicas:

```bash id="eoqf3x"
/w/nome-da-lista
```

Cada workspace possui:

* suas próprias tarefas
* suas próprias seções
* seu próprio ciclo de turno

Sem autenticação.

---

## 2. Seções

As tarefas devem ficar organizadas em seções como:

* 🌙 Fechamento
* 💖 Tarefas do Dia
* ⭐ Importantes

As seções devem aparecer em uma sidebar moderna e bonita.

---

## 3. Tasks

Cada tarefa deve possuir:

* título
* concluída ou não
* prioridade
* seção
* pin opcional

A interação deve ser extremamente fluida.

---

## 4. Turnos

Deve existir um botão:

```txt id="fx0hy9"
✨ Reiniciar Turno
```

Ao clicar:

* tarefas atuais são arquivadas
* um novo turno é criado
* tarefas recorrentes reaparecem automaticamente

---

## 5. Tarefas Recorrentes

O usuário pode marcar tarefas como recorrentes.

Essas tarefas:

* reaparecem automaticamente no próximo turno
* ficam salvas no Supabase

---

# Banco de Dados
